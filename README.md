# Campagne Nids de Poule

Application web mobile pour relever les nids de poule sur le terrain.

## Fonctionnalites

- Photo + GPS capture automatique au moment de la prise de vue
- Saisie rue, repere, gravite, surface
- Calcul automatique surface L x l
- Memorisation de la rue en cours
- Export ZIP (CSV + photos)
- Export KML pour Google Earth / QGIS
- Sauvegarde JSON pour reprendre la campagne

## Utilisation sur iPhone

1. Ouvrir l'URL Netlify dans **Safari**
2. Bouton Partager > **"Sur l'ecran d'accueil"**
3. Autoriser la localisation quand Safari le demande

## Utilisation sur Android

1. Ouvrir l'URL Netlify dans **Chrome**
2. Menu ... > **"Ajouter a l'ecran d'accueil"**
3. Autoriser la localisation quand Chrome le demande
